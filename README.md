# ⚙️ Monitory Barber - Painel Administrativo (Admin)

Módulo de gestão interna desenvolvido para o controle total da barbearia. Este painel permite que o barbeiro gerencie sua agenda, visualize faturamento e controle o fluxo de clientes em tempo real.

## 🚀 Acesse o Painel Admin
🔗 **Link:** https://tecnologia77.github.io/agenda-barbeariaDONAALD-S.admin.html/

🔑 **Acesso para Demonstração:**
* **Usuário:** `admin@teste.com`
* **Senha:** `teste123`

## 📊 Funcionalidades de Gestão
* **Dashboard de Agendamentos:** Visualização clara de todos os serviços marcados para o dia/semana.
* **Gestão de Horários:** Bloqueio e liberação de horários de forma instantânea.
* **Controle de Serviços:** Cadastro e edição de valores e tempos de execução.
* **Segurança:** Sistema de autenticação robusto via Firebase Auth.

## 🛠️ Tecnologias
* **Frontend:** JavaScript Avançado e Design Profissional (Dark Mode).
* **Banco de Dados:** Firebase (Persistência de dados e regras de segurança).
* **Infraestrutura:** Cloud Deployment com IP dedicado.
